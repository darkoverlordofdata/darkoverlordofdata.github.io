Exspresso

18 May 2013

An MVC Web Framework written in CoffeeScript.

Exspresso is a class based framework, in the style of php frameworks such as Kohana or CodeIgniter. Core classes act as an adapter layer to connect.js middleware, allowing you to focus on MVC design.
